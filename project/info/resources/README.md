# Resources

This directory is for documents you want the agent to extract knowledge from.

Examples:

 - Scientific articles
 - Technical manuals to guide the agent
   - E.g., for BAM files, the SAM specification is very helpful
 - Dataset schemas so it has an understanding of your data

Name files descriptively so agents can find them. You can also ask agents to extract the knowledge into markdown documents (either here or in `docs/research/`) for easier referencing later on.
