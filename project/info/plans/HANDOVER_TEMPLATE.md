# Handover Template

Use this when work is partially complete and another human or agent needs to continue without re-discovering the context.

## Task

What was the intended outcome?

## Current status

- What is done
- What is in progress
- What is not started

## Relevant files

- `path/to/file`: why it matters
- `path/to/other_file`: what changed or what still needs attention

## Decisions made

- Important technical or product decisions
- Tradeoffs that were chosen intentionally

## Open questions

- Anything unclear or blocked
- Any assumptions that should be confirmed

## Next steps

1. The next concrete action to take
2. The action after that
3. The smallest useful verification step

## Verification

- Commands already run
- Checks that passed
- Checks still needed

## Risks

- Known bugs
- Fragile areas
- Follow-up work worth doing soon
